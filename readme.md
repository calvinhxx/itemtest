# 📖 [Win UI Kit](https://aka.ms/WinUI/3.0-figma-toolkit)

## 1️⃣ Guidance & Charts
- Corner radius: 
  - 8px: Top level container 
  - 4px: In page container
  - 0px: intersect with other straight edges, maximized window.
- Layering & Material
  - Elevation: 类似qt中的z-order+阴影, 越顶层的窗口 阴影效果越明显.
  - Acrylic(透明), Mica(非透明), Smoke(模态背景)
  - Typography: 字体字号、对其、裁剪
  - Animation: win下的一套动画标准
  - Theme Color: light & dark两组主题色
  - Standard Screen Sizes: 一组标准屏幕大小的像素边界
  - Key App Window Sizes: App内关键size边界.
  - Surface Device Size.
